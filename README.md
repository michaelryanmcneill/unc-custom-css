UNC Custom CSS
=

A WordPress plugin that allows for Custom CSS to be easily added to WordPress.

## Origin

This plugin was forked from CTLT Dev's [Improved Simpler CSS](https://wordpress.org/plugins/imporved-simpler-css/).
